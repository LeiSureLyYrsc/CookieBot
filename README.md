# CookieBot

这是一个通过Bor处理加群请求的插件
目前处于Alpha测试状态
欢迎Pr(不是)

## How to start

1. generate project using `nb create` .
2. install plugins using `nb plugin install` .
3. run your bot using `nb run` .

由于此插件为单独插件，请修改Bot目录的`pyproject.toml`里的`plugin_dir`后将插件文件放入设定的文件夹载入
